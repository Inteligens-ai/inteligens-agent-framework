# When NOT to Use This Framework

This framework is intentionally opinionated.

## Not recommended for

- Fully autonomous production agents
- Real-time hard safety systems
- Ultra-low-latency pipelines
- Environments without human review

## Why

The Inteligens Agents Framework prioritizes:

- controlled execution
- auditability
- human-in-the-loop governance

If you need uncontrolled autonomous behavior, this is the wrong tool — by design.
