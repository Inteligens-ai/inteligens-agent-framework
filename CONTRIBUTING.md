# Contributing to Inteligens Agents Framework

Thank you for your interest in improving the Inteligens Agents Framework.

## How to contribute

### 🐛 Issues

- Use clear, reproducible descriptions
- Include logs when possible
- Label as bug / enhancement

### 🔧 Pull Requests

Before submitting:

- [ ] Code builds locally
- [ ] No secrets committed
- [ ] Documentation updated if needed
- [ ] Keep PRs focused and small

### 🧭 Philosophy

This project values:

- Human-in-the-loop safety
- Deterministic execution
- Clear agent responsibilities
- Production realism over hype

## Development setup

```bash
git clone <repo>
cd inteligens-agents-framework
python3 .agents/swarm/swarm_planner.py --task "test"
```
